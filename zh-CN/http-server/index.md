# http 服务器

基于 \Swoole\Http\Server 实现的协程HTTP服务, 框架层做了很好的封装, 用户按照传统的 MVC 方式编写代码, 就能获得协程带来的超高性能.

## 安装

```bash
composer require swoft/http-server
```

## 功能特色

- 基于 PSR-7 的 HTTP 消息实现
- 基于 PSR-15 的中间件
- `@Controller` 灵活的控制器注解
- `@RequestMapping` 灵活的路由注解
