# 配置组件

Swoft 配置由两部分组成，env 环境配置和 config 应用配置。

- `env` 一般配置一些和环境相关的一些参数，比如运行模式、资源地址
- `config` 一般用于配置应用级别的配置以及业务级别的配置

## 安装

```bash
composer require swoft/config
```
